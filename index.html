算法详情

插入排序

插入排序是一种简单直观的排序算法。它的工作原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

时间复杂度

最佳情况: O(n)

平均情况: O(n²)

最坏情况: O(n²)

空间复杂度

O(1)

private static void insertionSort(String[] arr) {
    for (int i = 1; i < arr.length; i++) {
        String key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j].compareTo(key) > 0) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

应用场景分析

check_circle_outline 小数据集

对于小型数据集，插入排序性能良好且实现简单。

check_circle_outline 几乎已排序数据

当数据已经接近排序状态时，插入排序接近线性时间表现。

check_circle_outline 内存受限环境

由于其O(1)的空间复杂度，非常适合嵌入式系统等内存受限环境。
